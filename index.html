Skor maksimal 100 poin. Nilai merupakan alat bantu penyaringan awal dan tetap memerlukan verifikasi administrasi serta lapangan.

Administrasi Maks. 30

Kondisi Teknis Maks. 40

Risiko Maks. 20

Penerima Manfaat Maks. 10

Dihitung dari unit terhuni: 1–24 = 2; 25–49 = 4; 50–99 = 6; 100–199 = 8; ≥200 = 10.

Arsitektur Integrasi SIRUMAH Kota Pangkalpinang

SIRUMAH Kota Pangkalpinang merupakan dashboard data, monitoring, dan layanan perumahan dengan semangat “Satu Data, Hunian Layak untuk Semua”. Sistem ini menghubungkan database sektoral perumahan, pemetaan, kebutuhan rumah, kualitas hunian, pengelolaan RUSUNAWA, layanan masyarakat, kolaborasi, serta laporan eksekutif dalam satu antarmuka.

Sumber Data Terintegrasi

Sumber data dibedakan berdasarkan karakter data berkala dan data transaksi.

Database Excel GitHub Tujuh sheet: PERUMAHAN, BACKLOG_KECAMATAN, RTLH_RLH_KECAMATAN, RTLH_INDIVIDU, RUSUNAWA_UNIT, RUSUNAWA_RETRIBUSI, dan RUSUNAWA_SARPRAS.
Google Sheets Layanan Form Layanan & Pengaduan dikirim melalui Google Apps Script dan direkap berdasarkan status penanganan.
GitHub Pages Menyajikan dashboard dan memuat workbook terbaru dari folder data tanpa menggunakan data contoh bawaan.

Tata Kelola, Akses, dan Penggunaan Data

Prinsip penggunaan dashboard untuk publik dan petugas internal.

Mode Publik Menjadi tampilan awal dan hanya memperlihatkan menu, kolom, kartu, serta rekap yang telah ditetapkan untuk masyarakat.
Mode Internal Diakses menggunakan PIN untuk membuka informasi teknis, data rinci, integrasi, laporan, cetak/PDF, dan ekspor CSV.
Ketersediaan Data Sheet yang belum terisi ditampilkan sebagai belum tersedia, bukan dianggap sebagai nilai nol resmi.
Pemetaan Data tanpa koordinat tetap masuk rekap, tetapi tidak dapat ditampilkan pada Peta Perumahan.
Prioritas Penanganan Skor prioritas merupakan alat bantu penyaringan awal dan tetap memerlukan verifikasi lapangan, status aset, kewenangan, serta kesiapan anggaran.
Perlindungan Data Data pribadi dan informasi internal tidak boleh ditempatkan pada repository publik atau diekspos melalui menu publik.

Integrasi Database Excel SIRUMAH

Dashboard memuat otomatis workbook terbaru dari folder data; nama sheet dan header divalidasi sebelum data diterapkan.

Generator Laporan Eksekutif SIRUMAH

Narasi otomatis menyesuaikan filter wilayah dan hanya menggunakan modul yang telah memiliki data.

Ringkasan Indikator Aktif

Indikator mengikuti filter wilayah; sumber yang kosong ditandai belum tersedia.

Catatan: Data SIRUMAH diperbarui secara berkala sebagai bahan informasi, monitoring, dan pendukung pengambilan keputusan. Penggunaan resmi tetap memerlukan verifikasi teknis, administrasi, dan kondisi lapangan.

Masuk Mode Internal

Masukkan PIN petugas untuk menampilkan informasi internal SIRUMAH.

PIN pada HTML statis hanya membatasi tampilan antarmuka. File Excel atau data yang ditempatkan pada repository publik tetap dapat diakses melalui alamat file.
